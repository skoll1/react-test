Tween api

  • animation:Object(一个),array(一组)需要执行的动画参数
  • paused:boolean 暂停动画
  • reverse:boolean 倒放动画
  • reverseDelay:number 倒放动画延迟
  • moment:number 设置当前时间上的时间，再次开始动画的起始时间
  • onChange:func 全局变动回调
  • ...其余属性暂时没遇到

animation

  • 和css有关
  • 动画有关参数
    • type:str to:正向，from:反向
    • duration: number 动画时间
    • delay: number 动画延迟
    • repeat: number 动画重复
    • repeatDelay: number 每次重复开始播放的时间
    • yoyo:boolean 重复执行返回动画
    • ease: string/func 缓动参数
